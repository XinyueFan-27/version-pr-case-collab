# version-pr-case-collab · 游戏版本舆情案例库

本仓库保留两个版本，互不覆盖：

- **[V1/](./V1/)** — 重做版（0+5 分析结构：游戏画像 / 事件复盘 / 原因追溯 / 官方处置 / 影响效果 / 案例启发；洛克王国 S2、三角洲共享监狱双样板）。
- **[original/](./original/)** — 初始版（重做前的原始网站，存档对照用）。

每个文件夹都是一套独立的纯前端站点。用浏览器打开对应文件夹的 `index.html` 即可；建议用本地静态服务器访问（`python3 -m http.server`），以便正常加载 JSON 数据。
